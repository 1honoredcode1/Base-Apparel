👕 Base Apparel Coming Soon Page

A responsive Coming Soon Landing Page built as part of a Frontend Mentor challenge.
The project focuses on creating a modern marketing page with an email subscription form and validation using HTML, CSS, and JavaScript.

🚀 Features

- Clean landing page layout
- Email subscription form
- Form validation (empty + invalid email)
- Responsive layout (mobile → desktop)
- Hero image section
- Modern typography and spacing

| Technology             | Purpose            |
| ---------------------- | ------------------ |
| **HTML5**              | Page structure     |
| **CSS3**               | Styling and layout |
| **JavaScript**         | Form validation    |
| **Flexbox / CSS Grid** | Layout alignment   |
| **GitHub Pages**       | Deployment         |
